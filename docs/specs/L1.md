# L1 — High-Level Requirements

## L1-1: Link Command

The CLI shall provide a root link command that symlinks a local npm library into an Angular workspace, updates tsconfig.json path mappings, and reports structured results.

## L1-2: Unlink Command

The CLI shall provide an unlink subcommand that reverses the linking process — removing the workspace symlink, unregistering the library globally, and cleaning up tsconfig.json path mappings.

## L1-3: Verify Command

The CLI shall provide a verify subcommand that checks whether a library is correctly linked by validating both the symlink target and the tsconfig.json path mapping values.

## L1-4: Architecture

The CLI shall use dependency injection, structured result types, and dedicated abstractions for npm process execution and tsconfig editing. Application logic shall have no direct console or filesystem coupling.
